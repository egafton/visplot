Telescope-specific pages
========================

Some telescopes have dedicated Visplot pages with pre-configured settings. These are accessible at ``https://www.visplot.com/[telescope_code]``. Here is a list of available telescope-specific pages:

- Nordic Optical Telescope (NOT): https://www.visplot.com/not
- William Herschel Telescope (WHT): https://www.visplot.com/wht
- Isaac Newton Telescope (INT): https://www.visplot.com/int
- Very Large Telescope (VLT): https://www.visplot.com/vlt

(Note: This list may not be exhaustive; check the main Visplot site for the latest additions.)
