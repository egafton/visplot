******************************************
Getting started
******************************************


If you want to try Visplot quickly, the easiest way is via the web interface.

1. Go to https://www.visplot.com/

2. Select a telescope in the :guilabel:`Configuration` and observing date

3. Load a predefined target or add your own

4. Generate a visibility plot or schedule

Within a few clicks, you should have a full overview of your observing night.

From here, you can:

* Follow the :doc:`tutorial </tutorial>` for a deeper walkthrough

* Explore the :doc:`web interface </webinterface>` in more detail

* Install Visplot locally using :doc:`custom installation </custom_install>`
